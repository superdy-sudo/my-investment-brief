## Moat Type
**Switching Costs + Network Effect (Marketplace)**

- Jira/Confluence ฝังลึกใน engineering workflow ขององค์กร — เปลี่ยนออกต้องย้ายข้อมูล, re-train ทีม, reconfigure integrations กับ CI/CD pipeline ทั้งหมด

- >\300,000 ลูกค้าใน 200+ ประเทศ ด้วยพนักงานแค่ ~12,000 คน — asset-light, high-margin model

## Investment Thesis (ระยะกลาง-ยาว 1-3 ปี)

**Why Atlassian Now:**
1. **AI monetization inflection** — Rovo (AI platform) ทำให้ลูกค้าที่ใช้ Rovo grow ARR ที่ rate ~2x vs ไม่ใช้ — นี่คือ tangible evidence ของ AI upsell ที่วัดได้
2. **RPO leading indicator** — Remaining Performance Obligations +37% YoY ($4.0B) ขณะที่ revenue +32% — แสดงว่า demand pipeline ยังแข็งแกร่งกว่า reported revenue
3. **Enterprise expansion** — ลูกค้าเซ็นสัญญาใหญ่ขึ้นและยาวขึ้น; Teamwork Collection bundle เพิ่ม ARPU ต่อ seat โดยไม่ต้องหาลูกค้าใหม่
4. **Free cash flow ยั่งยืน** — $1.4B FCF ต่อปีสม่ำเสมอ ขณะที่ revenue +20%/ปี = FCF กำลัง scale ขึ้น; $2.5B buyback program

**Valuation (ควรระวัง):**
- ราคา ~$85 บน market cap ~$22B
- EV/FCF (FY2025 base): ~15–16x — สมเหตุสมผลสำหรับ +20% growth business
- GAAP operating loss อยู่ (SBC สูง, R&D หนัก) — ต้องใช้ non-GAAP หรือ FCF เป็น lens หลัก
- หุ้นลง ~70%+ จาก ATH (~$480 ปี 2021) — ปัจจุบัน valuation ไม่ได้ฟุ้งเหมือนช่วง growth bubble

## Bear Case
"Software development workflow อาจ disrupted โดย AI coding agents — ถ้า AI เขียนโค้ดได้ทั้งหมด ความจำเป็นของ Jira project tracking อาจลดลง"

**ทำไม Bear Case นี้รับได้:**
- Atlassian ไม่ได้ขาย tool สำหรับแค่ developer — Jira Service Management (ITSM) และ Confluence (knowledge management) ใช้กับทุก function ในองค์กร ไม่ใช่แค่ engineering
- AI จะยิ่งทำให้ต้องการ coordination layer (Jira/Confluence) มากขึ้น ไม่น้อยลง — AI agents ก็ต้องการ tracking และ workflow orchestration
- Rovo คือ Atlassian's bet on AI ที่กำลัง work จริง (2x ARR growth signal)

## Risk Factors (Top 3 จาก 10-K)
1. **AI Disruption Risk** — AI coding tools (GitHub Copilot, Cursor) และ autonomous coding agents อาจลด dependency บน traditional project management หาก AI สามารถ automate workflows ได้ทั้งหมด (source: sources/TEAM/10-k-auto.md, Item 1A)
2. **Cloud Migration Execution** — Data Center to Cloud migration ยังดำเนินอยู่; failure to convert ลูกค้า Data Center ไป Cloud จะกดดัน ARPU growth และ ARR trajectory (source: sources/TEAM/10-k-auto.md, Item 1A)
3. **Competition จาก Microsoft** — Azure DevOps + Teams push harder ใน enterprise; ServiceNow แข่งใน ITSM; Linear แข่งใน modern developer workflow — Atlassian ต้องพิสูจน์ว่า AI (Rovo) เป็น differentiator ที่ยั่งยืน (source: sources/TEAM/10-k-auto.md, Item 1A)

## Kill Condition
ออกถ้า:
1. FCF margin ลดต่ำกว่า 20% เป็น 2 ไตรมาสติดต่อกัน (signal ว่า R&D/SBC cost structure ไม่ scale ตาม revenue)
2. Revenue growth ลดต่ำกว่า 15% YoY (จาก current trajectory ~24%) โดยไม่มีคำอธิบายชัดเจน — อาจหมายถึง cloud migration แผ่ว หรือ competition กัดเซาะ
3. Rovo AI adoption หยุดเติบโต หรือ ARR uplift จาก Rovo ลดต่ำกว่า 1.5x (จาก current ~2x) — แสดงว่า AI monetization thesis พัง
