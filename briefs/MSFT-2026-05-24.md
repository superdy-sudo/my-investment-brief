# MSFT — Microsoft Corporation
**วันที่:** 2026-05-24 | **ราคา:** ~$419 | **5d change:** 📈 +1.04%

## thesis (1 ประโยค)
Microsoft มี wide moat ที่แข็งที่สุดในกลุ่ม tech จาก enterprise lock-in + Azure momentum แต่ต้องระวัง capex AI ที่ยังไม่เห็น ROI ชัดเจน

---

## Fundamentals (keen)
- **Revenue:** $281.7B FY2025 (+15% YoY), Q1 FY2026 $77.6B (+18.4% — เร่งตัว)
- **Azure:** ทะลุ $75B annual run rate (+34% YoY), AI services +40% YoY ใน Q1
- **Operating Margins:** ไม่มีข้อมูลในไฟล์ (stable ตาม source แต่ไม่มีตัวเลข %)
- **FCF:** ไม่มีข้อมูลในไฟล์
- **Moat:** switching costs สูงมาก (M365/Azure/Dynamics), 75% desktop OS share, 400+ datacenters, stake ใน OpenAI
- **Risks:** ① Capex $34.9B/ไตรมาส ROI ยังไม่ชัด ② AWS/Google Cloud แข่งดุ ③ OpenAI exclusivity หมดแล้ว
