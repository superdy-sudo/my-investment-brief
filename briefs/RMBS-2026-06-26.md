## บริษัทนี้ทำอะไร?
Rambus มี 2 ธุรกิจ: (1) IP Licensing — 2,500+ patents ครอบคลุม memory architecture เก็บ royalty จาก Samsung, SK Hynix, Micron ทุกครั้งที่ผลิต DRAM (2) Fabless Chips — ออกแบบ DDR5 RCD และ HBM memory controller สำหรับ AI data center เปลี่ยนยากเพราะ patent ผูกกับ industry standard และ chip design ฝังในระบบ memory ของลูกค้า
